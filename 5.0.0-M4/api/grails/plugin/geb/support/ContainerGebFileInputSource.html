[Groovy] Class ContainerGebFileInputSource

      • grails.plugin.geb.support.ContainerGebFileInputSource 
      • class ContainerGebFileInputSource
extends java.io.File

        A File implementation specifically for assigning to the Geb FileInput module when using ContainerGebSpec. This will normalize the path for cross-platform compatibility (Windows paths with in Linux containers).
